# CLAUDE.md — 鲁班学派

鲁班学派是契约驱动的 AI Agent 角色分工协作体系。五个角色、七签功法、一份契约。修仙体系：求道→筑基→结丹→化神→渡劫。云笈是天书（功法总纲），不是角色。

## 角色技能强制触发

**行为基线**：每个角色技能已内嵌行为基线（三准绳 + 匠心入微 + 匠心标准 + 契约签），可直接加载使用。

## 协作流程

```
@问道 炼真言 → @司南 出契约 → @鲁班 执行编码 → [榫卯] 交付 → @狄公 审查 → 放行单 → @墨子 部署
  (求道)        (筑基)         ↑ 打回(≤3次)    ↓ 3次打回
                              └────────── @司南 仲裁 ──────────┘
```

## HARD-GATE 设计范围

`<HARD-GATE>...</HARD-GATE>` 标签用于**不可妥协的输出指令**——LLM 必须原样输出标签内文本/结构，不得改写、省略或自由发挥。

**五角色统一使用范围**（HG-2 定调）：

| 范围 | 说明 |
|------|------|
| 口头禅 | 五角色被召唤后第一句话，原样输出（防 LLM 自由发挥） |
| 司南门禁 | 契约三要素 TASK-ID + DONE + DON'T 缺一不可 |
| 鲁班门禁 | 无契约拒绝执行 |
| 狄公门禁 | 三张审查清单 + 放行/打回单格式 |

**不使用**: 开放式追问、性格描写、签文激活语——保留 LLM 表达灵活性。

详见 `docs/design/proposals/TASK-010-hard-gate-scope.md`。
